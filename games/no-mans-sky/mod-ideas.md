# No Man's Sky Mod Ideas

## Better System Scanning

- Auto-scan systems when your freighter enters them, as long as you have the planetary scanner
  installed.
- Add an exosuit capability that lets your freighter scan the system you are in even if the
  freighter is elsewhere. This would still require the planetary scanner on the freighter.

## Black Hole Tagging

- Surface black holes like planets for corvette autopilot.
- Display black holes in the GUI like planets on the discover or explore page.
- Allow the user to tag the black hole like they can tag planets in the GUI.

## Silent Visor

- Similar to what [Silent Scan](https://www.nexusmods.com/nomanssky/mods/690) tried to do. Turn off
  all sounds, or at least most sounds such as the humming, when using the analysis visor.
- Perhaps make it configurable via FOMOD.

## Smart Reticle

- Similar to [No Reticles](https://www.nexusmods.com/nomanssky/mods/4264), but instead hide the
  reticles until firing, then fade them when not firing. Ideally it could auto-detect nearby combat,
  such as ship battles or sentinels, and appear automatically without needing to be toggled by the
  user firing. For the multi-tool and exocraft, combat auto-detection would only apply to weapons.
  Tools like the mining laser and terrain manipulator would still need to trigger the reticle by
  firing.
- Ensure compatibility with [Reticles Color](https://www.nexusmods.com/nomanssky/mods/4193) and
  [No Pointer](https://www.nexusmods.com/nomanssky/mods/1354).

## System Sizer

- Similar to [gLonely Larger System Size](https://www.nexusmods.com/nomanssky/mods/4045), except
  configurable via FOMOD.
- Build in pulse drive speed increases that are also configurable via FOMOD. See
  [gShip Faster Ship Pulse Speeds](https://www.nexusmods.com/nomanssky/mods/3203) for inspiration.
