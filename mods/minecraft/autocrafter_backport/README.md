# AutoCrafter Backport Workspace

Standalone Forge 1.20.1 development workspace for the `autocrafter_backport`
mod unpacked from the Modrinth release jar.

## Notes

- Upstream artifact: `upstream/Autocrafter-1.0.0.jar`
- Remapped jar used for decompilation: `upstream/Autocrafter-1.0.0-named.jar`
- `src/main/java` was generated by remapping the shipped jar from SRG names to
  Mojmap-style names and then decompiling it with Vineflower.
- `src/main/resources` was extracted from the upstream jar.

## Useful Commands

- `./gradlew compileJava` - type-check sources
- `./gradlew runClient` - launch the Forge client dev run
- `./gradlew runServer` - launch the Forge server dev run
- `./gradlew build` - build the mod jar
- `./gradlew spotlessApply` - format Java sources
